什么是MCP？

MCP 是一种开放协议，通过标准化的服务器实现，使 AI 模型能够安全地与本地和远程资源进行交互。

此列表重点关注可用于生产和实验性的 MCP 服务器，这些服务器通过文件访问、数据库连接、API 集成和其他上下文服务来扩展 AI 功能。

[!NOTE]
关于本地 🏠 和云 ☁️ 的区别：

  • 当 MCP 服务器与本地安装的软件通信时使用本地服务，例如控制 Chrome 浏览器。
  • 当 MCP 服务器与远程 API 通信时使用网络服务，例如天气 API。
